🎯 VISION & KONZEPT
Die Brücke zwischen zwei Welten
Welt 1: Millionen Menschen suchen Fokus-Tools. FocusFlight beweist: Aviation-Ästhetik macht süchtig ($3.99/mo, 4.8★).
Welt 2: Tausende VATSIM-Piloten fliegen 24/7 realistische Routen um die Welt — unsichtbar für Nicht-Simmer.
Welt 3: Flight-Sim-Piloten investieren Tausende in Hardware, verdienen aber nichts mit ihrem Können.

SkyFocus Live verbindet alle drei: Eine wunderschöne Focus-App, die echte VATSIM-Flüge auf einer Live-Karte zeigt — und Sim-Piloten ermöglicht, ihre Cockpit-Streams live zu teilen und echtes Geld zu verdienen (Tips, Subs, Featured Flights).
🎯
FocusFlight-Level UX
Immersives Onboarding, Aurora-Glow, Glassmorphism, Premium Dark UI. Mindestens so gut wie FocusFlight ($3.99/mo App). 46 extrahierte Frames als UI-Referenz vorhanden.
📡
VATSIM Live Data
Echte Flugsimulator-Piloten auf Mapbox GL. 1,000-2,000+ gleichzeitige Flüge. Callsign, Route, Altitude, Speed, Aircraft Type. Updates alle 15s. ATC-Stationen sichtbar.
💰
Piloten verdienen Geld
Sim-Piloten streamen ihren Cockpit-View. User zahlen Tips, Subscriptions, oder wählen "Featured Flights". Twitch für die Flugsim-Welt — aber elegant in eine Focus-App integriert.

🎬 PILOT STREAMING — DAS HERZSTÜCK
Twitch für Flight Sim — aber besser
Auf Twitch gehen Flugsim-Streams unter. Auf SkyFocus Live sind sie die Stars. Jeder VATSIM-Pilot kann seinen Cockpit-Stream direkt an seinen Live-Flug auf der Karte koppeln. User tippen auf den Flieger → sehen den Cockpit-Stream → geben Tips. Der Pilot fliegt seinen normalen VATSIM-Flug und verdient dabei Geld.
🎥
Stream Setup
Pilot nutzt OBS Studio (gratis) → streamt an SkyFocus RTMP-Server → Stream wird automatisch mit VATSIM-Position verknüpft via CID. Kein kompliziertes Setup — ein Stream Key, fertig.
📐
Multi-Perspektive
Pilot kann bis zu 3 Kamera-Feeds streamen: Cockpit (Primary), Window Seat, External Cam. User wählt seine bevorzugte Perspektive. MSFS24 SimConnect für automatische Kamera-Umschaltung.
